# Donation Management System

A Java Swingâ€“based application for managing donations between donors and associations.

## Features
- Donors can create, update, view, and delete donations
- Associations can view and collect available donations
- Role-based dashboards
- MySQL database integration

## Technologies Used
- Java
- Java Swing
- MySQL
- Git

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Add MySQL Connector JAR to the project
4. Run the `Main` class
